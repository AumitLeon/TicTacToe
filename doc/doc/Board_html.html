  class Board
</h1>

<section class="description">

    <p>Creation of the matrix. The <a href=“Board.html”>Board</a> is a seperate class so that it is more modular. This board can be modified to fit the needs of users for other projects, and can be expanded or made smaller from its standard 3x3 size. To remain a box, the overall size must remain a perfect square (4. 9. 16. 25…).</p>
